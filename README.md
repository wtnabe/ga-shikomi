# GAShikomi

Google Analytics API wrapper for CLI written in Ruby.

## Installation

Now, You can fetch general ga data.

```ruby
gem 'ga-shikomi'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ga-shikomi

## Usage

type

    shikomi help
